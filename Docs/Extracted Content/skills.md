## Rewrite guidance

- Do not translate `verb` bodies one by one.
- Create `RprSkillPrototype` with activation type, target type, cost, cooldown, requirements, effects, and legalStatus.
- Add named effect handlers only when generic data cannot express a skill.
- Mark franchise-specific names as placeholders until Phase 18.
- Start Phase 12 with six skills: one melee, one projectile, one beam/channel, one buff, one movement utility, one noncombat utility.

## Test cases

- Skill activation rejects insufficient energy.
- Skill activation rejects cooldown bypass.
- Skill activation rejects missing race/class/form requirements.
- Client target data is treated as a request, not truth.
- Skills with visual prediction reconcile when server denies activation.
